Planetary texture maps in this directory are a mix of sources.

LEGACY BASE MAPS (from Solar System Scope, CC BY 4.0):
  sun.jpg, mercury.jpg, venus.jpg, earth_day.jpg, mars.jpg, uranus.jpg, neptune.jpg, stars_milky_way.jpg, saturn_ring.png and others.
  https://www.solarsystemscope.com/textures

PHOTOREALISTIC / ENHANCED TEXTURES:
  earth_clouds.jpg, jupiter.jpg, saturn.jpg, moon.jpg  (and supporting maps in the folder)
  Some generated via xAI Grok Imagine for better PBR detail in the 3D scenes.

NEW PLUTO MODEL:
  pluto_diffuse.png, pluto_normal.png, pluto_specular.png — high-quality PBR textures from user-provided model, used with MeshStandardMaterial (diffuse + normal + roughness) on high-res sphere in the Atlas, with correct axial tilt.

Legacy originals for reverted assets (Earth day, Mars) and ring are the classic Solar System Scope versions.
Extra generated maps (normals, night, etc.) and /legacy/ folder are present for reference but not all are loaded after reverts.

See source code (src/engine/atlas.ts, stages.ts) for exact material usage per body.
